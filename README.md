# 📝 min-Excalidraw

![Excalidraw Logo](https://github.com/excalidraw/excalidraw/blob/master/dev-docs/static/img/logo.svg)

## 概要

このリポジトリは **Excalidraw** をセルフホストで動作させる、最小限のテスト用サンプルです。

- **React + Vite** で構築
- Excalidrawの描画・読み込み専用表示が可能
- 公式パッケージのみ利用、シンプルな構成
- 表示ページと同階層のExcalidrawファイルを表示

## 使い方

1. リポジトリをクローン
2. `npm install`
3. `npm run dev`
4. ブラウザで `http://localhost:5173` を開く

1. リポジトリをクローン
2. `npm install`
3. `npm run build`
4. ウェブサーバーに、出力されたdist内ファイルをアップロード
5. Excalidrawファイルを編集
6. 必要に応じ、main.jsxのEXCALIDRAW_FILEを編集
7. ブラウザでウェブページにアクセス
