# 📝 min-Excalidraw

![Excalidraw Logo](https://raw.githubusercontent.com/excalidraw/excalidraw/master/public/logo192.png)

## 概要

このリポジトリは **Excalidraw** をセルフホストで動作させる、最小限のテスト用サンプルです。

- **React + Vite** で構築
- Excalidrawの描画・読み込み専用表示が可能
- 公式パッケージのみ利用、シンプルな構成

## 使い方

1. リポジトリをクローン
2. `npm install`
3. `npm run dev`
4. ブラウザで `http://localhost:5173` を開く
